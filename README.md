# Autonomous mobile robotics
Incorporate the following elements of mobile robotics onto a [Thymio robot](https://www.thymio.org/):

* Vision
* Motion Control
* Filtering
* Global Navigation
* Local Navigation

Full project details can be viewed in `final_report.ipynb`.
